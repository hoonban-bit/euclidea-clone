# Euclidea Clone: 수동 스모크 테스트 가이드

엔진의 대규모 리팩토링 후 배포하거나 코드를 제출하기 전에, 기여자는 기하학 엔진의 핵심 기능이 잘 작동하는지 수동으로 검증해야 합니다. 이 "스모크 테스트"는 유닛 테스트가 놓칠 수 있는 실제 사용자 경험의 버그(Regression)를 잡기 위해 설계되었습니다.

테스트를 진행하려면 개발 서버(`npm run dev`)를 실행하고 브라우저에서 애플리케이션을 엽니다.

---

## 1. DAG 연쇄 삭제 테스트 (종속성 그래프)
**목표:** 부모 도형을 삭제했을 때, 그 도형에 의존하는 모든 자식 도형들이 재귀적으로 파괴되는지 확인합니다.

**순서:**
1. **Line Tool(선 도구)**을 선택합니다. 서로 교차하여 "X" 자를 만드는 두 개의 선을 그립니다.
2. 두 선이 교차하는 지점에 주황색 교차점이 자동으로 나타나는지 확인합니다.
3. **Circle Tool(원 도구)**을 선택합니다. 가운데 교차점에 스냅(Snap)하여 클릭한 뒤 밖으로 드래그하여 원을 그립니다.
4. **Eraser Tool(지우개 도구)**을 선택합니다.
5. 처음에 그렸던 선 중 **단 하나**만 클릭하여 지웁니다.

**기대 결과:**
클릭한 선이 사라집니다. 선이 사라졌으므로 교차점도 즉시 사라집니다. 교차점이 사라졌으므로 (그 점을 중심으로 하던) 원도 즉시 연쇄적으로 사라집니다. 지우지 않은 나머지 선 하나는 온전하게 남아 있어야 합니다.

---

## 2. 스냅핑 및 자동 교차점 테스트
**목표:** 순수 수학적 교차가 스냅 가능한 점들을 올바르게 생성해 내는지 확인합니다.

**순서:**
1. **Circle Tool(원 도구)**을 선택하여 원을 하나 그립니다.
2. 첫 번째 원과 겹치도록(벤 다이어그램처럼) 두 번째 원을 그립니다.
3. 두 원의 가장자리가 교차하는 두 지점에 주황색 교차점이 자동으로 나타나는지 확인합니다.
4. **Line Tool(선 도구)**을 선택합니다.
5. 위쪽 교차점 근처로 마우스를 가져갑니다. 파란색 "스냅 링"이 나타나는지 확인한 후 클릭하여 선을 시작합니다.
6. 아래쪽 교차점 근처로 마우스를 가져갑니다. 파란색 링이 나타나는지 확인한 후 클릭하여 선을 마칩니다.

**기대 결과:**
그려진 선이 두 원이 겹치는 수학적 교차점을 정확하게 가로지르며 완벽하게 이등분해야 합니다.

---

## 3. 불변성 테스트 (Undo / Clear)
**목표:** 행동 히스토리 배열이 이전 상태를 변형(Mutate)시키지 않고 보드의 상태를 올바르게 추적하는지 확인합니다.

**순서:**
1. **Clear** 버튼을 클릭하여 보드가 초기 상태(레벨 1 기본 점)인지 확인합니다.
2. 선을 하나 그립니다.
3. 원을 하나 그립니다.
4. 지우개 도구를 사용하여 선을 지웁니다.
5. 우측 상단의 **Undo** 버튼을 클릭합니다.

**기대 결과:**
Undo를 한 번 누르면 지웠던 선이 다시 나타나야 합니다. 한 번 더 누르면 원이 사라져야 합니다. 세 번째로 누르면 처음에 그렸던 선이 사라지고 게임 시작 초기 상태만 남아야 합니다.

---

## 4. 레벨 검증 테스트
**목표:** 사용자가 목표에 도달하기 위해 어떤 구체적인 경로를 거쳤든 상관없이, `LevelVerifier`의 의미론적 헬퍼 함수가 기하학적 형태를 바탕으로 승리 조건을 성공적으로 감지하는지 확인합니다.

**순서:**
1. **Clear** 버튼을 클릭하여 레벨 1을 초기화합니다.
2. **Circle Tool(원 도구)**을 선택합니다. 왼쪽 기본 점에서 시작해 오른쪽 기본 점까지 드래그하여 원을 그립니다.
3. 오른쪽 기본 점에서 시작해 다시 왼쪽 기본 점으로 드래그하여 두 번째 원을 그립니다.
4. **Line Tool(선 도구)**을 선택합니다. 왼쪽 기본 점에서 시작해 두 원이 만나는 *위쪽* 교차점까지 선을 그립니다.
5. 오른쪽 기본 점에서 시작해 *위쪽* 교차점까지 선을 그립니다.

**기대 결과:**
마지막 선을 그리고 마우스 클릭을 떼는 순간, 화면 상단 중앙에 커다란 녹색 "Level Complete!" 알림창이 나타나야 합니다.

*추가 테스트:* Clear를 클릭한 후 같은 과정을 반복하되, 이번에는 선을 *아래쪽* 교차점 방향으로 그립니다. 이 경우에도 성공적으로 레벨이 클리어되어야 합니다.
