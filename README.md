# Euclidea Clone

This is an open-source clone of the popular geometric construction game [Euclidea](https://www.euclidea.xyz/).

The objective of this project is to build an interactive application that allows
users to construct geometric figures using a straightedge and compass, solve
puzzles by discovering the most elegant solutions, and learn fundamental
geometric concepts interactively.

Please check `GOALS.md` for a comprehensive list of goals and features planned
for this project.
